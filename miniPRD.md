# 엑셀 기반 다국어 회화 프로그램 (속청)

## 1. 개요
- 엑셀 파일에서 다국어(한국어/영어/중국어) 문장을 읽어와 음성과 자막으로 학습하는 프로그램
- Edge-TTS 기반의 고품질 음성 합성 사용
- 사용자 맞춤형 학습 설정 지원

## 2. 주요 라이브러리
- **음성 합성**: edge-tts,비손실 음성 합성(mp3사용 금지)
- **오디오 처리**: 
  - pygame.mixer (실시간 재생)
  - soundfile, librosa (오디오 파일 처리)
  - ffmpeg (오디오 속도/볼륨 조절)
- **UI**: tkinter
- **이미지 처리**: PIL, moviepy
- **데이터 처리**: pandas, openpyxl
- base 폴더에 음성 파일 2개, 엑셀 파일, 
- 스크린 가로 1080, 비율 16:9

## 3. 주요 기능
- 화면 사이즈, 배경색, 기본자막 폰트 사이즈 등 디폴트 설정 가능
- 초기화면, 카운트다운,학습화면, 휴식화면, 통계화면 등 디폴트 설정 가능
- 모든 화면을 한개의 창에 연속적으로 표시(별도창 사용하지 않음)
- 엑셀 기본언어는 a열 언어 영어, b열 언어 한국어, c열 언어 중국어
- 엑셀에서 헤더 없이 총 600문장인데, 문장 수 자동인식
- 엑셀에서 a,b,c열 언어가 변경되도 사용 가능하게 설정
- 화면 순서 : 시작화면, 카운트다운, 학습화면, 브레이크, 학습통계, 시작화면
- 화면은 한개의 창에서 연속적으로 표시
- 
### 3.1 초기화면(모든 항목값 기억)
- 학습 범위 설정 (시작/끝 행), 랜덤 재생 여부(6단계 난이도 조절)
- 순위 언어선택 : 자막 표시 순서, 1~3순위 언어 선택, 1~3개 선택
- 순위 언어별 음성 재생 횟수 설정(1-3회)
- 순위 언어+재생횟수 세트로 표시
- 언어별 음성 선택
  - 영어: 9개 음성,기본값 en-US-SteffanNeural
  - 한국어: 2개 음성,기본값 ko-KR-SunHiNeural
  - 중국어: 2개 음성,기본값 zh-CN-XiaoxiaoNeural
- 재생 설정
  - 언어별 재생 속도
  - 문장 간 간격
  - 다음 문장 대기 시간
  - 자막 대기 시간(음성보다 자막이 먼저 표시되는 시간)

### 3.2 학습화면
- 다크모드/브라이트 모드 선택
- 전체 테두리 표시, 파란색, 6픽셀
- 진행 상황 표시
  - 현재 문장 번호(우상단 : 001, 파란색)
  - 좌상단 문구 : 한글 속청 30일 영어 귀가 뚫린다! 아이보리색
  - 총 소요 시간(하단 중앙1)
  - 문장당 평균 시간(하단 중앙2)
  - 초기화면 버튼(하단 중앙3)
- 3단 자막 디스플레이(초록색)
  - 선택된 언어 순서대로 표시(1~3개 언어 선택)
  - 현재 음성 재생 중인 자막 하이라이트(아리보리색)
  - 선택 언어수에 따라 자막 폰트 사이즈 조절
  - 선택 언어수에 따라 상하 중앙 정렬
  - 좌우 중앙 정렬, 문장이 길어지면 자동 줄바꿈 
  - 영어 자막 폰트 사이즈 160pt, 한국어 자막 폰트 
-  사이즈 120pt, 중국어 자막 폰트 사이즈 100pt
-  자막 표시 방법
- 1번 자막, [자막대기] 음성 재생, [문장간격]
- 2번 자막, [자막대기] 음성 재생, [문장간격]
- 3번 자막, [자막대기] 음성 재생, [문장간격]
- 3번 자막 : 음성 미재생시는 2번 자막과 함께 처리
* 자막유지 옵션 : 2,3번 자막은 다음 문장 음성 완료시 까지 유지

### 3.3 재생 기능
- 언어 우선순위 설정
  - 1~3순위 언어 지정(자막 표시 순서)
  - 자막 표시후 음성 재생(자막 대기 시간에 따라 음성 재생 시간 조절)
  - 순위 언어별 음성 반복 횟수 설정
- 재생 간격 제어
  - 문장 내 언어 간 간격
  - 문장 간 간격
  - 자막 선행 표시 시간
- 음성 설정
  - 언어별 재생 속도 (1.0~4.0배)
  - 영어 고속 재생 시 볼륨을 크게 보정

### 3.4 부가 기능
- 카운트다운
  - 학습 시작전 3초 카운트 다운
  - 드럼 사운드 효과, 3,2,1 카운트 효과
  - 문구 내용 : 이 영상은 몸에 좋은 wav 파일로 제작했습니다. 타이핑 효과
- 브레이크
  - 설정된 문장 수마다 휴식(20문장이면 20문장후 휴식)
  - 휴식 시간 커스터마이징, 기본 5초
- 학습 통계
  - 총 학습 시간
  - 문장당 평균 시간
  - 학습한 문장 수
  - 3초간 표시후, 초기화면으로 이동

## 4. 디자인 설정
### 4.1 기본 테마/화이트모드/기타 모드
- 배경색: #1e1e1e (다크모드)
- 자막 색상:
  - 기본: #4CAF50 (녹색)
  - 하이라이트: #FFFFF0 (아이보리)
  - 진행 표시: #2196F3 (파란색)

### 4.2 폰트 설정
- 카운트다운: Helvetica, 250pt, bold
- 메시지: Helvetica, 63pt
- 자막: AppleSDGothicNeo, 160pt
- 진행 상황: AppleSDGothicNeo, 56pt
