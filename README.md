# 도파민 대충영어 : 생활영어 600문장, 여행영어 800문장, 천일문 500문장
- 한국어 중국어 일본어 베트남어 다국어 학습 프로그램
- 2-4배속 속청 기반 영어 귀뚫기 프로그램

한국어 영어 중국어 일본어 베트남어 회화 프로그램, 엑셀 문장을 음성과 자막으로 학습하는 파이썬 스트림릿 프로그램입니다.

## 주요 기능

- 엑셀 파일 기반 다국어 문장 학습
- Edge-TTS 기반 고품질 음성 합성
- 사용자 맞춤형 학습 설정
- 다크모드 지원
- 학습 통계 제공

## 설치 방법

1. Python 3.11.1 이상 설치
2. 필요한 패키지 설치:
   ```bash
   pip install -r requirements.txt
   ```
3. ffmpeg 설치 (음성 속도 조절에 필요)
   - macOS: `brew install ffmpeg`
   - Windows: [ffmpeg 다운로드](https://www.ffmpeg.org/download.html)

## 사용 방법

1. 엑셀 파일 준비
   - `en600new.xlsx` 파일에 학습할 문장 입력
   - A열: 영어, B열: 한국어, C열: 중국어
   - 헤더 없이 문장만 입력

2. 프로그램 실행
   ```bash
   python Edge600-501.py
   ```

3. 학습 설정
   - 학습 범위 설정: 시작 행, 끝 행 지정
   - 언어 순서 설정: 1~3순위 언어 선택
   - 음성 설정: 언어별 음성과 속도 선택
   - 재생 설정: 문장 간 간격, 자막 대기 시간 등 설정

4. 학습 시작
   - "학습 시작" 버튼 클릭
   - 카운트다운 후 학습 시작
   - 설정된 간격마다 휴식
   - 학습 완료 후 통계 확인

## 주요 설정 항목

### 학습 범위 설정
- 시작 행: 학습을 시작할 엑셀 파일의 행 번호
- 끝 행: 학습을 끝낼 엑셀 파일의 행 번호

### 언어 설정
- 1~3순위 언어: 자막 표시 순서 지정
- 언어별 반복 횟수: 각 언어의 음성 재생 횟수

### 음성 설정
- 영어 음성: 9개 음성 중 선택
- 한국어 음성: 2개 음성 중 선택
- 중국어 음성: 2개 음성 중 선택
- 언어별 재생 속도: 1.0~4.0배

### 재생 설정
- 문장 간 간격: 다음 문장으로 넘어가는 간격
- 자막 대기 시간: 음성 재생 전 자막 표시 시간
- 브레이크: 설정된 문장 수마다 휴식

## 기타 정보

- 음성 파일 형식: WAV (무손실)
- 지원 언어: 영어, 한국어, 중국어
- 화면 모드: 다크모드 (기본)
