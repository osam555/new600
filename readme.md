# 도파민 대충영어

머리가 좋아지는 영어 학습 앱입니다.

## 주요 기능

- 14개국어 지원 (한국어, 영어, 중국어, 일본어, 베트남어, 필리핀어, 태국어, 러시아어, 우즈베크어, 몽골어, 네팔어, 미얀마어, 인도네시아어, 캄보디아어)
- 영어 19개국 48명의 다양한 음성 모델 제공
- 자동 반복 학습
- 휴식 시간 설정
- 자막 크기 및 색상 커스터마이징
- 음성 속도 조절

## 설치 방법

1. Python 3.11 이상 설치
2. 필요한 패키지 설치:
```bash
pip install -r requirements.txt
```

## 실행 방법

```bash
streamlit run en600_st_app.py
```

## 사용 방법

1. 학습할 언어 선택 (최대 3개)
2. 음성 모델 및 속도 설정
3. 학습 범위 선택
4. 자막 스타일 설정
5. 학습 시작

## 주의사항

- 음성 파일은 임시 디렉토리에 저장되며 자동으로 삭제됩니다.
- 학습 중에는 브라우저를 닫지 마세요.
- 음성 재생에 문제가 있는 경우 브라우저를 새로고침해보세요.
